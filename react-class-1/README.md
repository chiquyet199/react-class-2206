
## Step
** Must install `node` to run `npm`
1 - run `npm init react-app app-name`
2 - run `cd app-name`
3 - run `npm start`

## Step to run on new code
1 - `git pull`
2 - `cd react-class-1`
3 - `npm install` 
4 - `npm start`
